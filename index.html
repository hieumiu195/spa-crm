☁️ Lưu Dữ Liệu Cloud – Dùng Trên Mọi Máy

Sau khi thiết lập, dữ liệu sẽ tự động lưu lên Google Sheets của chị — mở trên điện thoại, máy tính nào cũng được, không bao giờ mất.
📋 Hướng dẫn thiết lập (làm 1 lần duy nhất):
1. Mở sheets.google.com → tạo bảng tính mới, đặt tên "Spa CRM Data"
2. Click menu Tiện ích mở rộng → Apps Script
3. Xoá hết code cũ, dán code từ file SpaCRM_CloudScript.gs vào
4. Click Triển khai → Triển khai mới → Ứng dụng web
5. Đặt: Thực thi là "Tôi", Ai có quyền truy cập "Bất kỳ ai"
6. Click Triển khai → Sao chép URL → Dán vào ô trên → Kết Nối ✅
